# 🛡️ Avoid Note

**Avoid Note** is a community-driven intelligence platform designed to identify, report, and scan for deceptive ad patterns and malicious URLs.

## 🚀 Features

- **Safety Scanner:** Real-time URL analysis against known malicious databases.
- **The Evidence Locker:** A public database of community-reported dark patterns (Fake close buttons, forced redirects, etc.).
- **Type-Safe Database:** Built with Prisma and PostgreSQL for high-speed reporting.
- **Modern UI:** A minimalist, "Legit" design built with Tailwind CSS v4 and Lucide icons.

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS v4
- **Database:** Prisma ORM + Vercel Postgres
- **Notifications:** Sonner
- **Icons:** Lucide-React
